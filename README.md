## shooter-di-rx

Небольшой Unity-проект, упор на UniRX и Zenject

### Реализовано

*Bootstrap*

**Player**

* WASD передвижение (Unity Input System) через Transform
* Автовыбор ближайшего врага в радиусе атаки при остановке
* Автоатака с частотой из конфига, нанесение урона
* Конфиг (`PlayerConfig`): скорость передвижения, скорость атаки, урон, радиус атаки
* (Дополнительно) визуализация цели линией `LineRenderer`
* (Дополнительно) реализовано движение камерой от первого лица

**Enemies**

* Патруль между заданными точками
* HP, получение урона, смерть, респавн через время
* Лимит максимального количества активных мобов на уровне
* Конфиг (`EnemyConfig`): HP, время респавна, скорость, лимит мобов
* (Дополнительно) Защита от лишних мобов при старте сцены

**UI**

* Счётчик убитых врагов (реакция на события смерти)

### Технологии

* **Zenject**
* **UniRx**

### Использованные паттерны

* Dependency Injection: модули собираются в `PlayerInstaller` / `EnemyInstaller`
* Observer: события ввода, смерти врагов, свободных слотов и респавна оформлены как `IObservable<T>`
* Component-based composition: логика разделена на небольшие компоненты (Input/Move/Attack/Health/Respawn/Patrol/Collection)
